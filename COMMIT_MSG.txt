1. Remaining Dart code expressions (arithmetic, method calls, array access)
2. Broken ICU fragments (messages starting with }, ], or ))

## Root Cause
Previous cleanup scripts missed patterns like:
- Arithmetic operations in time ranges (e.g., "10:00-18:00" detected as "0-1")
- Broken ICU message fragments from incomplete code removal
- Unbalanced braces/brackets from partial deletions

## Solution - Two-Phase Nuclear Cleanup

### Phase 1: Remove Dart Code Patterns
Detected and removed keys containing:
- Dart interpolation: ${...}
- Array/List access: [0], .length, as List
- Method calls: .join(), .split(), .format()
- Conditional operators: ?, ??, !=, ==, <=, >=
- Arithmetic operations: + digit, - digit, * digit
- Code blocks: setState(), return;, print()

Result: Deleted 551 keys with Dart code across all ARB files

### Phase 2: Remove Broken ICU Fragments
Detected and removed keys with:
- Messages starting with }, ], or )
- Unbalanced braces/brackets
- Closing brackets/braces without opening

Result: Deleted 53 broken fragment keys across all ARB files

### Phase 3: Synchronization
Retained only common keys across all 7 language files
Result: Removed 76 language-specific keys
