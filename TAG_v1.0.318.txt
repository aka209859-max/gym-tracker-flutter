## Root Cause
v1.0.314-317 ARB cleanup removed 8,492 keys, but left behind:
1. AppLocalizations calls in CONST contexts (compile-time)

3. Methods without BuildContext trying to use localization

Dart cannot evaluate AppLocalizations.of(context) at compile time.
Const contexts require literals, not runtime values.
