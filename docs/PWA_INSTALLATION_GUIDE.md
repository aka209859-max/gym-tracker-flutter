## 🎯 PWAとは？

**Progressive Web App（プログレッシブウェブアプリ）**は、Webサイトとネイティブアプリの良いとこ取りをした技術です。

### ✅ PWAのメリット

| 項目 | ネイティブアプリ | PWA | 通常のWebサイト |
|------|----------------|-----|----------------|
| **インストール** | App Store/Google Play必須 | ワンタップでホーム画面に追加 | 不要 |
| **容量** | 50-200MB | 5-10MB | 0MB（キャッシュのみ） |
| **更新** | ストア審査待ち（数日～数週間） | 即座に反映（リロードのみ） | 即座に反映 |
| **オフライン対応** | ✅ 対応 | ✅ 対応 | ❌ 非対応 |
| **プッシュ通知** | ✅ 対応 | ✅ 対応（準備済） | ❌ 非対応 |
| **ホーム画面アイコン** | ✅ あり | ✅ あり | ❌ なし |
| **全画面表示** | ✅ 対応 | ✅ 対応 | ❌ ブラウザバー表示 |
| **開発コスト** | 高額（Android + iOS別開発） | 低額（1つのコードで両対応） | 最安 |

### 🎯 FitSync PWAの特徴

- ✅ **軽量**: わずか5MB程度の容量
- ✅ **高速**: リリースモードで最適化済み
- ✅ **オフライン対応**: 一度訪問した画面はオフラインでも表示可能
- ✅ **プッシュ通知準備済**: 混雑度アラート機能（Phase 3で実装予定）
- ✅ **自動更新**: アプリを開くたびに最新版に自動更新

## 📱 インストール手順（iPhone / iOS）

### 必要なもの
- iPhone（iOS 11.3以降推奨）
- Safari ブラウザ（**Chrome/Firefoxは非対応**）

### ステップバイステップ手順

#### 1️⃣ **Safariで FitSync を開く**

```
URL: https://5060-i1wzdi6c2urpgehncb6jg-5c13a017.sandbox.novita.ai
```

> 💡 **重要**: 必ず**Safari**で開いてください。Chrome や Firefox では PWA インストールができません。

#### 2️⃣ **共有ボタンをタップ**

- 画面下部の中央にある **共有アイコン**（□に↑矢印）をタップ

<details>
<summary>📸 スクリーンショット例を見る</summary>

```
┌─────────────────────┐
│  FitSync            │
│  ジムマップ          │
│                     │
│  [現在地ボタン]      │
│                     │
└─────────────────────┘
      ↓
   [共有ボタン] ← ここをタップ
```

</details>

#### 3️⃣ **「ホーム画面に追加」をタップ**

- 共有メニューが表示されます
- 下にスクロールして **「ホーム画面に追加」** を見つける
- タップして選択

> 💡 **見つからない場合**: 共有メニューの一番下までスクロールしてください

#### 4️⃣ **名前を確認して「追加」をタップ**

- アプリ名: **FitSync**（変更可能）
- アイコン: 自動設定済み
- 右上の **「追加」** をタップ

#### 5️⃣ **ホーム画面から起動**

- ホーム画面に **FitSync アイコン** が追加されます
- タップして起動すると、**全画面表示**でアプリのように動作します

### ✅ インストール成功の確認

- ✅ ホーム画面に FitSync アイコンが表示される
- ✅ 起動時に Safari のアドレスバーが表示されない（全画面）
- ✅ 通常のアプリと同じように動作する

## 🤖 インストール手順（Android）

### 必要なもの
- Android スマートフォン（Android 5.0以降推奨）
- Chrome ブラウザ（推奨）または Edge ブラウザ

### ステップバイステップ手順

#### 1️⃣ **Chrome で FitSync を開く**

```
URL: https://5060-i1wzdi6c2urpgehncb6jg-5c13a017.sandbox.novita.ai
```

#### 2️⃣ **自動インストールプロンプトを確認**

- ページを開くと、画面下部に **「ホーム画面に追加」** バナーが表示される場合があります
- 表示された場合は **「追加」** をタップ → 完了！

#### 3️⃣ **手動インストール（バナーが表示されない場合）**

1. 画面右上の **︙（3点メニュー）** をタップ
2. **「ホーム画面に追加」** または **「アプリをインストール」** をタップ
3. ポップアップで **「インストール」** をタップ

**方法B: アドレスバーのアイコン**

1. アドレスバーの右側に **「+」アイコン** が表示される場合があります
2. タップして **「インストール」** を選択

#### 4️⃣ **ホーム画面から起動**

- ホーム画面またはアプリドロワーに **FitSync アイコン** が追加されます
- タップして起動すると、**全画面表示**でアプリのように動作します

### ✅ インストール成功の確認

- ✅ ホーム画面に FitSync アイコンが表示される
- ✅ 起動時に Chrome のアドレスバーが表示されない（全画面）
- ✅ タスクスイッチャーで独立したアプリとして表示される

## 🔧 トラブルシューティング

### Q1: 「ホーム画面に追加」が見つからない（iPhone）

**原因**: Safari の共有メニューがカスタマイズされている

**解決策**:
1. 共有メニューを一番下までスクロール
2. **「アクションを編集」** をタップ
3. **「ホーム画面に追加」** を有効化して保存

### Q2: インストールプロンプトが表示されない（Android）

**原因**: PWA の最小要件を満たしていない、またはブラウザが対応していない

**解決策**:
1. Chrome ブラウザを最新版に更新
2. 手動インストール方法（3点メニュー → ホーム画面に追加）を試す
3. それでも表示されない場合は、ブックマークに追加して使用

### Q3: アイコンが表示されない

**原因**: キャッシュの問題

**解決策**:
1. アンインストールして再インストール
2. ブラウザのキャッシュをクリア
3. 端末を再起動

### Q4: オフラインで動作しない

**現状**: Phase 2 では基本的なキャッシュ機能のみ実装済み

**Phase 3 で改善予定**:
- 完全なオフライン検索機能
- バックグラウンド同期
- より高度なキャッシュ戦略

## 💡 よくある質問（FAQ）

### Q: PWA は安全ですか？

**A: はい、非常に安全です。**

- ✅ HTTPS 通信で全データ暗号化
- ✅ ブラウザのセキュリティ機能を継承
- ✅ 不要な権限要求なし（位置情報のみユーザーが許可）
- ✅ データは端末内に保存（個人情報は収集しない）

### Q: アンインストール方法は？

**iPhone**: 
1. FitSync アイコンを長押し
2. 「Appを削除」をタップ

**Android**: 
1. FitSync アイコンを長押し
2. 「アンインストール」または「削除」をタップ

**PC**: 
1. Chrome で `chrome://apps` を開く
2. FitSync を右クリック → 削除

### Q: データ通信量はどのくらい？

**A: 非常に少ないです。**

- 初回ロード: 約 3-5MB
- 2回目以降: ほぼゼロ（キャッシュ利用）
- GPS 検索1回: 約 10-50KB
- 1ヶ月の想定通信量: 10-50MB程度
