# 💳 請求先アカウントの確認方法

## 📍 左上のメニューから確認

```
Google Cloud Console:
1. 左上の "≡" (ハンバーガーメニュー) をクリック
2. メニューをスクロールして以下を探す:
   
   "お支払い" または
   "Billing" または
   "課金"
   
3. クリック
```

### スクリーンショット

このメニュー画面をスクリーンショットで共有してください。

---

## 別の確認方法

### プロジェクト設定から確認

```
Google Cloud Console:
1. 左メニュー → "IAM と管理"
2. "設定" をクリック
3. "請求先アカウント" セクションを確認
   
   【設定済み】
   ✅ 請求先アカウント名が表示
   
   【未設定】
   ❌ "請求先アカウントなし"
   ❌ "リンク" ボタンが表示
```

---

## 📸 スクリーンショットを共有してください

以下のいずれかを共有してください:

1. 左メニュー（≡）の全体
2. "お支払い" メニューの画面
3. "IAM と管理" → "設定" → "請求先アカウント" の部分
