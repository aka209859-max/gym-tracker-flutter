# 💳 請求先アカウントの確認

## 🚨 APIキーが新しくても403エラー = 請求先アカウント未設定の可能性

---

## 確認方法

### Google Cloud Console で確認

```
1. Google Cloud Console を開く
   https://console.cloud.google.com/

2. プロジェクト "GYM MATCH" を選択

3. 左上のメニュー（≡）をクリック

4. "お支払い" または "Billing" をクリック

5. 請求先アカウントの状態を確認:
```

### パターンA: 請求先アカウントが設定されている
```
✅ 請求先アカウント名が表示される
✅ クレジットカード情報が登録されている
→ OK、別の原因
```

### パターンB: 請求先アカウントが未設定
```
❌ "請求先アカウントをリンク" ボタンが表示される
❌ "このプロジェクトには請求先アカウントがありません"
→ これが原因！
```

---

## 請求先アカウントが未設定の場合

### 設定方法
```
1. "請求先アカウントをリンク" をクリック
2. "請求先アカウントを作成" を選択
3. 国/地域: 日本
4. クレジットカード情報を入力
5. 確認して設定

注意:
- 無料枠内なら料金は発生しません
- クレジットカードは本人確認のためのみ
- Gemini API: 1日1,500リクエストまで無料
- Places API: 月28,500リクエストまで無料
```

---

## ⚠️ 重要

Google Cloud の多くのAPIは、**無料枠であっても請求先アカウントの登録が必須**です。

理由:
- 本人確認
- 無料枠を超えた場合の保護
- 不正利用防止

---

## 📸 確認してほしいスクリーンショット

以下をスクリーンショットで共有してください:

```

左メニュー → お支払い（Billing）

画面のスクリーンショット:
- 請求先アカウントが設定されているか
- または "リンク" ボタンが表示されているか
```
