## 🔧 考えられる原因と対策

### 原因1: GitHub Actionsビルドが失敗

**対策**:
1. GitHub Actionsのログを確認
2. エラーメッセージを共有
3. 修正して再ビルド

### 原因2: TestFlightアップロードが失敗

**症状**: ビルドは成功しているが、TestFlightに表示されない

**対策**:
1. GitHub Actionsの最後のステップ「Upload to App Store Connect」を確認
2. App Store Connect (https://appstoreconnect.apple.com) で手動確認
3. 必要なら手動でアップロード

### 原因3: 古いビルドがインストールされている

**症状**: TestFlightに新しいビルドがあるが、アプリは古いまま

**対策**:
1. TestFlightアプリで明示的に**ビルド246**を選択
2. 「インストール」または「更新」をタップ
3. アプリを再起動

### 原因4: キャッシュ問題

**症状**: 新しいビルドをインストールしても変更が反映されない

**対策**:
1. アプリをアンインストール
2. TestFlightから再インストール
3. デバイスを再起動
